Image Optimization

We love seeing images in websites! They make websites more interesting and appealing, but they can also make them slow. A typical mobile phone camera will take an image that is over 2000px wide and 1-3MB in size. Just to put that into perspective a typical resolution for a 23" monitor is about 2000px and on a typical cellular connection 2 MB will take 5-10 seconds to download! Can you imagine what would happen if you placed several of these large images in your site? Many users, especially those with a slow internet connection or on a mobile device, would experience a severe drop in the loading speed of your site. For many it would quickly become unacceptable.

We can easily use CSS to adjust the sizes at which those images appear, but the browser will still have to download the whole image before it can shrink it. A better solution is to use an image editing program and change its size to just what we need.

If you have ever tried to load an image heavy website and it took forever, you understand the need to optimize images for a better user experience. Optimizing images for websites is necessary and easy, you do not need to have expensive software like Photoshop. Do not forget though that image optimization is still about image quality, creating images that are to small in size still can also ruin the user experience. Focus on decreasing image size to a reasonable size for the websites design.

Assignment

  1. Learn

    Now days there are lots of image editing software at your disposal. An older online service that is still useful is: picresize.com If you have never resized an image before you should watch the following video that teaches you about the process using the pic resize website.

    How to Optimize Images for the Web Online for Free

    You are not required to follow or complete the examples in this article, but you should spend some time reviewing Optimize Images for Web by Martin Williams. These are advanced concepts that a graphic designer, web designer, and web developer should be aware of.

  2. Practice

    Your goal for this assignment is to download and resize this picture. Please resize this image to 250 x 313 pixels; this is 250 pixels wide by 313 pixels tall. You can use any image editing software at your disposal including these additional online resources:

    • Pixlr eXpress: super easy image edits.
    • Pixlr Editor: advanced online image editing.
    • Resize Image Network: simple step by step image editing.

  3. Upload

    Once you have resized and downloaded your image you will need to add it to your Glitch website. In the Glitch Editor click on the assets folder in the left sidebar; it is the top option in your list of website files. Next click on the Upload an Asset button to add this image to your Glitch site. Once the image is added to your asset gallery, please click on it to display its URL. Copy this URL for the next step.

    Please add a link on your home page to this image with a target value of _blank. Normally we place image URL's in an img tag but we can also provide them as the href value in a (link) tags. This will load just the image in a new tab for easy grading.

  4. Complete Assignment

    Follow the instructions in canvas to prove you have completed this assignment. To get the URL to your live Glitch site click on Show in the upper left and then New Window. When the new tab loads with your live site copy the URL from the address bar. Make sure the URL you submit for grading is to your home page that contains a link to the image.

    Grading:

    Do not submit the URL to your Glitch profile or your Glitch editor, it must be the URL to the home page on your live site. There must be a link on this page to your resized image uploaded to Glitch.
