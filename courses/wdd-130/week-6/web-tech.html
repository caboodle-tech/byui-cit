Web Technology

So far in this course we have primarily focused on web design and web development. Today we want to spend some time discussing web technology, or the programming, networking, and tools that support design and development as a whole. This assignment aims to broaden your view of web fundamentals and introduce you to some advanced roles in design and development. It will also help you better understand similar careers and skills from fields besides Web Development.

Assignment

  1. Computer Programmer

    Career Explorer has a fantastic online article on Computer Programmers. Please read the overview section (page 1) for this assignment. You may want to return later to study all the sections.

    As you study about Computer Programmers pay special attention to the additional training, skills, and responsibilities they have compared to a Web Designer or Web Developer. Remember, the terms Computer Programmer or Programmer are generic umbrella terms meant to cover several advanced job roles.

  2. Site Reliability Engineer

    Open Source (supported by RedHat) has a fantastic online article on Site Reliability Engineers. Please read the article for this assignment. You may want to return later to study all the sections.

    In summary a good Sight Reliability Engineer is a software engineer with a good administration background or a highly skilled system administrator with knowledge of coding and automation. Their responsibility is not web design or web development, but to provide teams with the tools and processes that help them excel at development.

  3. Network & Hardware

    Another aspect of software development and design to consider is the infrastructures that support them. It takes servers, workstations, customized hardware (computers) sometimes, and various specialized software tools to support all the jobs that fall under the Web Fundamentals, Software Development, and Applications Development umbrellas.

    One of the learning outcomes for this course is to help you decide if a Web Design and Development degree is the educational path you should pursue. Please review some of the following job roles and keep in mind that a Web Design and Development degree may help with some but is probably not the degree you should pursue for these careers:

    • Network Architect
    • Network Administrator
    • Database Administrator
    • Database Architect
    • Cytotechnologist
    • Technical Artist
    • Information Security Analyst
    • IT Security Consultant
    • Security Engineer
