The 3 Web Pillars

This is a summary page meant to bring together your Week 4 reading assignment Web Designer vs. Web Developer and your Week 6 reading assignment Web Technology. Together these assignments have taught you about the 3 Web Pillars:

  • Technology
    • Advanced Programming
    • Hardware & Software Tools
    • Networks & Workstations
  • Design
    • Wire Framing
    • Graphic Design
    • Typography & Color
    • User Experience / User Interface
  • Development
    • Programming (Coding)
    • Data Storage (Databases)
    • User Experience

Please review the reading assignments and all the material they refer you to, then take the 3 Web Pillars quiz in Canvas.
