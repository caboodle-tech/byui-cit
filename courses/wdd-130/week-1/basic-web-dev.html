Basic Web Development Concepts

To start the semester off we need to make sure you understand basic web development concepts at a high level. We will build on this foundation throughout the semester.

Assignment

Please watch the following video below before our first day of class. As you watch the video ponder and consider these questions:

  • What is the Internet?

  • What is a browser?

  • What is a server?

  • What languages are used to create websites?

  • What is a client and who does it talk to?

Note any questions, comments, or concerns you have as you watch the video so we can discuss them together in class.
