Web Designer vs. Web Developer

This is your first assignment studying some of the jobs roles that fall under the Web Fundamentals umbrella. We are starting with Web Designers and Web Developers because their job roles are often confused or combined in industry. This assignment aims to show you that Web Designers and Web Developers are two distinct job roles requiring their own skill sets.

  3. Overview

    A Web Designer focuses on the design (look and feel) of a site. Web Designers have various titles that usually include the word designer in their name, such as User Experience (UX) Designer or User Interface (UI) Designer. Other tiles include Digital Content Curator, Usability Designer, Digital Director, and so on. These advanced titles are meant to differentiate the responsibilities and expertise of a designer. A UX/UI Designer for example probably has experience in one or more wire framing tools and software applications like Photoshop.

    A Web Developer focuses on creating the actual website or application. In other words a Web Developer makes the design work in the real world. Web Developers primarily work with HTML, CSS, and JavaScript. A more advanced developer would also work with a backend programming language like PHP, a database language of some kind like MySQL, and possibly a pre-processor of some kind like LESS or SASS for CSS. Developers with advanced skill sets often go by different titles like Software Developer, Backend Developer, Frontend Developer, Full Stack Developer, and so on. These advanced titles are meant to differentiate the responsibilities and expertise of a developer. Software Developers for example know how to program software for various applications, not just the web.

    In industry it is common to mix titles to denote a combination of skills and responsibilities. For example a UX/UI Developer is both a designer and developer in one role. When searching for jobs read the job descriptions carefully, the job tile is sometimes incorrect or does not accurately reflect your expected skills and responsibilities.
