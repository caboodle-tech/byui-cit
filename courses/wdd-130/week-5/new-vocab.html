New Vocabulary

So far in this course you have been exposed to a lot of new vocabulary. Most of this vocabulary refers to things you will use often like HTML elements and CSS rules. There has been terms though that have not come up as consistently. Please study the following terms so you can better understand web design and development. You will be quizzed at the end of the week on these terms.

Assignment

  1. General Terms

    Make sure you have read, understood, and memorized the following terms:

    Alpha Version
    An early form of a new piece of software, application, or website, that is being tested to check that it is working correctly so far. Often alpha versions are incomplete and buggy. Typically employees of the company that are developing it will perform the testing.
    Beta Testers
    Real users that test in a new piece of software, application, or website in a production environment running on the same hardware, networks, and so on, as the final release would be.
    Beta Version
    A piece of software, application, or website, that is at the second stage of development. During beta customers or dedicated testers are often asked to use it to see if there are any problems that have not been corrected.
    Development Release
    Also known as a dev release or referred to as released to dev. This is a deployable software package, application, or website that is the culmination of several iterations (beta versions and tests). A dev release is a different kind of beta version that could become a production release.
    Live Site
    A production software, application, or website that is accessible to the anticipated group of users. In the context of this class a live site refers to your website being publicly available on the internet for everyone to view.
    Optimization
    Optimization is the action of making the best or most effective use of a situation or resource. In the context of this class, optimization refers to images, CSS files, and JavaScript files that are optimized to speed up page loading and lower the transferred data size from our website to users.
    Production Release
    Also known as prod release and PR, or referred to as in production and in prod. This is a stable release which has passed all verification checks and tests. If there are any remaining bugs they are considered as acceptable.
    SEO
    SEO, or earch engine optimization, is the process of maximizing the number of visitors to a particular website by ensuring that the site appears high on the list of results returned by a search engine.
