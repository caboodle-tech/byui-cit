Colors

Colors are one of the key cornerstones to proper User Experience (UX). The colors you choose for your website will have a massive impact on how users perceive and interact with your site. As you will learn in this course colors play several key roles, for example they help with visual contrast or the readability of your site, they can also set a tone or mood helping a visitor feel or react a certain way. Colors are a subtle, and sometimes not so subtle, way to communicate something to the visitors of your site. You also need to keep user accessibility in mind when working with colors. Your sight might look great to you but a visually impaired user could have a drastically different opinion, especially if clashing colors or dim colors make the site unusable for them.

CSS Color In Practice

The following video provides you with a brief introduction to creating CSS code for colors. During this course you will be shown several times how to do this. If you have not practiced with the following CSS rules, remember that background is a shorthand property allowing you to provide multiple background rules at once, color is one of the properties you can assign with background. To specifically apply only a background color you should use the background-color rule instead. To change the font (text) color use the color rule. You can do an internet search for W3Schools CSS Colors if you would like to learn more about the CSS color rules.

CSS Color Palettes & Wheels

There are hundreds of color picking and palette tools available on the internet or as a software application. Some of these tools are loaded with ads, others offer unique features seeking to stand out from the crowd. You are welcome to do an internet search for CSS Color Pickers, CSS Color Wheel, or Color Palette to find a color picker you like. If you would like to demo one of these tools right away we have gathered a list of 3 of the most popular tools below. We would recommend using one of these tools for this course. Please pay careful attention to the notes we have added:

  • Adobe Color Wheel: A feature rich color wheel made by Adobe, the creators of Photoshop. The tool is free to use but some features require you to have an Adobe account.
  • Coolors: An easy to use color palette generator, simply press space to cycle colors. Warning when the site first load there will be a lot of flashing, close the dialog box and press space to stop.
  • Colormind: Similar to Coolors with very little bells and whistles. Simple and straight forward to use.
