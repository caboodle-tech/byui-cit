HTML Cards

This week you will design the wireframe for your about us page. Next week you need to be ready to build the HTML for that page. In class your instructor will demonstrate the basics of HTML cards but this assignment will help teach you the concepts in greater depth. There is nothing to turn for this assignment but you will be quizzed on the concepts taught here.

Assignment:

  1. Study

    W3Schools has a simple to follow tutorial on HTML Cards split into 2 parts. You will need to study the HTML part and the CSS part. Please note that the CSS part is demonstrating W3Schools style guide. You can use the tutorials to see how all the styles work but you will need to use your inspector to view the actual CSS code.

    Free Frontend has built a collection of 85 CSS Cards that you might find helpful when its time to make the HTML for your about us page.

  2. Practice and Question

    Make sure you understand what you studied from the previous section. Practice building your own HTML cards to get ready for next week. If you have questions or concerns make sure to bring them up in class so you instructor can demonstrate the answer.

  3. Complete

    There is nothing to turn in for this study assignment but you will be quizzed on this material later in the week.
